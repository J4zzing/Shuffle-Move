Ce programme utilise cette page-ci pour télécharger la dernière version mise à jour. Vous pouvez retrouver les précédentes versions sur ce wiki
. Cliquez sur « Obtenir la dernière version » pour télécharger la version la plus récente dans votre répertoire d'installation. Il faudra la dézipper après avoir fermé ce programme pour terminer la mise à jour. Lorsque le téléchargement aura pris fin, le dossier où la dernière version a été téléchargée s'ouvrira si possible.
